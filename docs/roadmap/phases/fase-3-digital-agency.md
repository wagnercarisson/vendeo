# 📍 Fase 3 — Digital Agency

## 🎯 Objetivo

Eliminar o trabalho manual do lojista no processo de marketing.

O Vendeo passa a não só criar conteúdo, mas também:

👉 executar e distribuir

---

## 🧠 Problema atual

Mesmo com conteúdo pronto, o lojista ainda precisa:

* postar manualmente
* copiar texto
* escolher horário
* lembrar de publicar

Isso gera:

* fricção
* esquecimento
* inconsistência

---

## 💡 Estratégia

Transformar o Vendeo em um fluxo completo:

👉 criar → organizar → publicar

---

## 📦 Features desta fase

### 1. Distribuição 1-Clique (MVP)

Permitir que o usuário publique conteúdo rapidamente.

Fluxo:

* botão “Postar”
* abre Instagram ou WhatsApp com conteúdo pronto

📌 Objetivo:

* reduzir fricção
* acelerar uso

📌 Importante:

* começar com abordagem semi-automática
* não depender de integração complexa inicialmente

---

### 2. Agendamento de Campanhas

Permitir que o lojista planeje a semana inteira.

Exemplo:

* segunda → oferta
* quarta → combo
* sexta → promoção forte

📌 Integração natural com:

* planos semanais existentes

📌 Diretrizes:

* interface simples
* foco em facilidade (não complexidade)

---

### 3. Publicação Automática (Evolução)

Permitir que o Vendeo publique automaticamente.

📌 Possibilidades:

* integração com Meta (Instagram/Facebook)
* publicação agendada

📌 Observações:

* depende de permissões e APIs externas
* implementar após validação do MVP

---

### 4. Link na Bio Dinâmico

Criar uma página automática com:

* campanhas recentes
* ofertas ativas
* conteúdos gerados

📌 Objetivo:

* centralizar o tráfego
* aumentar conversão

📌 Conceito:

* mini “vitrine digital” atualizada automaticamente

---

## 🚫 Fora do escopo (nesta fase)

* criação de anúncios pagos (Ads)
* análise avançada de dados
* recomendações inteligentes profundas

---

## 📊 Métricas de sucesso

* número de postagens realizadas via Vendeo
* uso de agendamento
* redução de tempo entre criação e postagem
* aumento de consistência de publicação

---

## 🎯 Resultado esperado

O Vendeo passa de:

👉 ferramenta de criação

para:

👉 sistema completo de execução de marketing
