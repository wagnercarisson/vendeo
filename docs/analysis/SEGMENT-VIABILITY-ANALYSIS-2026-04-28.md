## 📊 ANÁLISE DETALHADA: ADEGAS + MERCEARIAS

### 1. CONTEXTO DO MERCADO

#### Adegas (Especialização: Bebidas)
- **Universo Brasil:** ~15.000 lojas (fontes SEBRAE, 2025)
- **Ticket médio:** R$ 45-120 por compra
- **Frequência:** Semanal-quinzenal (recompra constante)
- **Margens:** 25-35% (distribuição + marca própria)
- **Concorrência:** ALTÍSSIMA (supermercados, e-commerce, distribuidoras)

#### Mercearias (Generalista: Mix de produtos)
- **Universo Brasil:** ~45.000 lojas (pequeno/médio)
- **Ticket médio:** R$ 60-180 por compra
- **Frequência:** Semanal (cliente "do bairro")
- **Margens:** 20-30% (compra no distribuidor)
- **Concorrência:** ALTÍSSIMA (mercados do bairro, supermercados chain, apps)

**TOTAL TAM:** ~60.000 lojas com pressão competitiva imediata

---

### 2. POR QUE ADEGAS + MERCEARIAS SÃO IDEAIS PARA VENDEO

#### 🔴 Problema Principal: Falta de Visibilidade Social
**Situação atual:**
- 95% das adegas/mercearias NÃO possui presença social estruturada
- Quando tentam (Instagram), é manual, desorganizado, sem estratégia
- Clientes descobrem loja por acaso, vizinhança (sem amplificação)
- E-commerce e supermercados chain "roubam" clientes por FALTA de engajamento social

**Impacto no ROI:**
```
Antes (sem social):
├─ 100 clientes base no bairro
├─ Churn estimado: 30-40% ao mês (mudança, perda para concorrência)
└─ Crescimento: ZERO (dependente de footfall)

Depois (com Vendeo):
├─ 100 clientes base
├─ Social ads amplificam 2-3x o alcance
├─ Retenção melhora: +20-30% (familiarity)
├─ Novo cliente: +15-25% ao mês (discovery)
└─ Ticket médio: +10-15% (urgência/impulso das campanhas)

IMPACTO ESTIMADO: +30-50% faturamento em 3 meses
```

#### 🟢 Oportunidade 1: Produtos Simples, Hierarquia Clara
**Cenários de Campanha Tipicamente Fáceis:**

| Tipo | Exemplo | Complexidade | Vendeo Fit |
|------|---------|--------------|-----------|
| **Promoção Semanal** | "Cerveja 350ml R$2,50" | BAIXA | ✅ Perfeito |
| **Combo** | "Cerveja + Chopp + Petisco" | BAIXA | ✅ Perfeito |
| **Reposição** | "Voltou! Achocolatado Y" | BAIXA | ✅ Perfeito |
| **Sazonalidade** | "Drinks para o fim de semana" | MÉDIA | ✅ Bom |
| **Urgência** | "Última caixa! Vinho X importado" | MÉDIA | ✅ Bom |
| **Mix/Ofertas Complexas** | "Compre 3 itens e ganhe desconto" | ALTA | ⚠️ Desafiador |

**Aplicabilidade:** ~85% das campanhas de adegas/mercearias são BAIXA ou MÉDIA complexidade = **perfeito para Vendeo v1**.

#### 🟢 Oportunidade 2: Planejamento Estratégico Natural
**Adegas/Mercearias JÁ PENSAM EM SEMANAS:**

```
Fluxo Típico de Lojista:
┌─────────────────────────────────────────────────┐
│ SEGUNDA-FEIRA (Abastecimento)                   │
│ - Recebe fornecimento do distribuidor            │
│ - Define promoções pra semana                    │
│ - "Vou colocar cerveja em promoção"              │
└─────────────────────────────────────────────────┘
                      ↓
┌─────────────────────────────────────────────────┐
│ QUARTA/QUINTA (Pico de Movimento)               │
│ - Movimento começando a aumentar                 │
│ - "Vou colocar um combo pra quinta"              │
└─────────────────────────────────────────────────┘
                      ↓
┌─────────────────────────────────────────────────┐
│ SEXTA/SÁBADO/DOMINGO (Pico Total)               │
│ - Movimento máximo                              │
│ - "Vou fazer promoção relâmpago"                │
│ - Clima muda → "Bebida fria = venda quente"     │
└─────────────────────────────────────────────────┘
                      ↓
┌─────────────────────────────────────────────────┐
│ SEGUNDA (Reinicia Ciclo)                        │
└─────────────────────────────────────────────────┘
```

**Por que é ideal para Weekly Plan:**
- Lojista JÁ PENSA EM CALENDÁRIO
- Sabe quais dias vender + e quais vender -
- Planejar "3-5 campanhas por semana" é NATURAL
- Não precisa educá-lo, só automatizar o pensamento dele

#### 🟢 Oportunidade 3: Consistência Visual = Diferencial
**Problema:** Adegas/mercearias "se confundem" no social

```
Cenário Sem Vendeo (Redes Sociais Caóticas):
├─ Fototelefone de produto (quality ruim)
├─ Post texto genérico "Aproveite!"
├─ Sem design, sem identidade
└─ Cliente: "Qual loja é isso? (Não engaja)"

Cenário Com Vendeo (Identidade Visual):
├─ Cada campanha tem a MARCA da loja
├─ Cores, layout, tom reconhecível
├─ Cliente: "Reconheço! Essa é a bodega do João"
└─ RETENÇÃO +30% (brand recall)
```

**Vendeo + Consistent Visual Signature = Vantagem Competitiva Real**

> "A consistência visual é o que diferencia adegas/mercearias bem-posicionadas de concorrentes caóticos"

### 4. COMPLEXIDADE TÉCNICA DO MOTOR VISUAL

#### Tipos de Imagens Esperadas em Adegas

```
1. PRODUTO ISOLADO (80% dos casos) ✅ FÁCIL
   ├─ Garrafa de cerveja em fundo branco
   ├─ Caixa de suco
   ├─ Pote de bebida
   └─ Visual Reader: Detecção > 95%

2. MÚLTIPLOS PRODUTOS (15% dos casos) ✅ MÉDIO
   ├─ Combo: 2-3 bebidas + petisco
   ├─ Oferta: "Compre 2 leve 3"
   └─ Visual Reader: Agrupamento > 85%

3. LIFESTYLE (5% dos casos) ⚠️ DESAFIADOR
   ├─ Pessoa segurando bebida gelada
   ├─ Mesa com drinks
   └─ Visual Reader: Contextualização > 70%
```

**Score: 8/10** — Maioria dos casos é fácil; casos complexos têm fallback.

## 💰 MODELO DE PLANOS PARA ADEGAS

### FREE (5 campanhas/mês)
- **Objetivo:** Onboarding, validação
- **Limite:** 5 campanhas/mês (suficiente para testar)
- **Sem Weekly Plan**
- **Sem histórico de dados**
- **ROI:** Validar que Vendeo funciona

### BASIC (Ilimitado + Weekly Plan)
- **Objetivo:** Uso regular, operacional
- **Limite:** Campanhas ilimitadas
- **Weekly Plan:** ✅ Incluído (4 variações/semana)
- **Sugestões de IA:** Boas práticas (não IA total)
- **Preço:** R$ 79-99/mês
- **ROI:** Payback em 1 mês

### PREMIUM (IA Orquestradora)
- **Objetivo:** Automação total
- **Limite:** Campanhas ilimitadas + agendamento
- **Weekly Plan:** ✅ Incluído
- **IA Orquestradora:** ✅ Sim (clima, sazonalidade, dados históricos)
- **Analytics:** Survey → métricas de conversão real
- **Preço:** R$ 199-249/mês
- **ROI:** Payback em 2-3 semanas

- **Alvo:** 500+ adegas em SP + RJ + MG
- **Duração:** 8 semanas

- **Alvo:** 5.000+ adegas Brasil
- **Duração:** 6 meses
- **Objetivo:** Scale profitável
- **Métrica:** MRR (Monthly Recurring Revenue) R$ 100k+
