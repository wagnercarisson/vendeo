## ✅ DECISÃO REVISADA

**Implementar subsegmentação com CHECK constraints no DB + 4º subsegmento bebidas:**

### 1. Schema com Valores Controlados

```sql
-- CHECK constraint para category (valores permitidos)
ALTER TABLE stores 
ADD CONSTRAINT check_category_values 
CHECK (category IN (
  'bebidas_alcoolicas',
  'mercearia'
));

-- CHECK constraint para subcategory bebidas
ALTER TABLE stores 
ADD CONSTRAINT check_subcategory_bebidas 
CHECK (
  category != 'bebidas_alcoolicas' OR 
  subcategory IN (
    'adega', 
    'loja-bebidas', 
    'distribuidor', 
    'emporio-cervejas',  -- ← NOVO (recomendação Mercer)
    'outro', 
    NULL
  )
);

-- CHECK constraint para subcategory mercearia
ALTER TABLE stores 
ADD CONSTRAINT check_subcategory_mercearia 
CHECK (
  category != 'mercearia' OR 
  subcategory IN (
    'mercadinho-bairro',
    'minimercado',
    'hortifruti',
    'emporio-gourmet',
    'sacolao',
    'outro',
    NULL
  )
);
```

**Resultado:** DB **rejeita automaticamente** valores não controlados, mesmo via API direta.

## 🎯 GAP IDENTIFICADO: Empório de Cervejas

### Por Que Adicionar 4º Subsegmento?

**Perfil:** Empório de Cervejas / Beer Store / Craft Beer Shop

| Característica | Valor |
|---------------|-------|
| **Foco** | Cervejas artesanais/importadas (não vinhos) |
| **Público** | Cervejeiros entusiastas, 25-45 anos, ABC1/B |
| **Ticket médio** | R$ 80-150 (entre loja comum e adega) |
| **Diferencial** | Curadoria em CERVEJA (como adega, mas em cerveja) |
| **Tom** | Descolado, craft culture (não formal como adega) |

### Por Que NÃO Cabe em "Adega"?

- ❌ Foco exclusivo em cerveja (não vinho)
- ❌ Tom menos formal (mais jovem, craft culture)
- ❌ Público diferente (25-35 vs. 40-60)

### Por Que NÃO Cabe em "Loja de bebidas"?

- ❌ Não vende "gelada rápida" — vende EXPERIÊNCIA
- ❌ Curadoria é central (não conveniência)
- ❌ Ticket 3× maior

### Exemplos Reais

- "Hop House" (São Paulo)
- "Cervejaria Artesanal" (Florianópolis)
- "Beer Store" (Porto Alegre)
- "Empório das Cervejas" (Curitiba)

**Cobertura:** +5% do mercado (nicho crescente em capitais)
