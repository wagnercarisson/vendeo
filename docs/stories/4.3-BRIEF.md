## 🎯 Objetivo

Implementar o **Motor 3 (Visual Composer)** — o motor de geração de variações compositivas que recebe a direção criativa e retorna 4 layouts distintos mas coerentes para o usuário escolher.

**O que ele faz:**
- Recebe `ImageProfile` (Motor 1) + `CreativeDirection` (Motor 2) + `VisualSignature` + `CampaignData`
- Gera **exatamente 4 variações** de layout (CompositionSpec)
- Cada variação define: posições de elementos, hierarquia visual, tipografia, espaçamentos, badges
- Garante que variações são **visualmente distintas** (30% diferença mínima)

**O que ele NÃO faz:**
- Não analisa imagens (isso é Motor 1)
- Não decide direção criativa (isso é Motor 2)
- Não renderiza artes finais PNG/JPG (isso é Motor 4)
- Não aplica cores da Visual Signature (isso é Motor 4)

## 🔒 Regras de Validação

### Regra 1: Canvas Boundaries

Todos os elementos DEVEM estar dentro do canvas 1080×1350:

```typescript
function validateBounds(area: {x, y, width, height}): boolean {
  return (
    area.x >= 0 &&
    area.y >= 0 &&
    area.x + area.width <= 1080 &&
    area.y + area.height <= 1350
  );
}
```

### Regra 3: 30% Distinctness entre Variações

**Garantir que as 4 variações são visualmente distintas.**

```typescript
function validateDistinctness(variations: CompositionSpec[]): boolean {
  const threshold = 0.3; // 30% diferença mínima
  
  for (let i = 0; i < variations.length; i++) {
    for (let j = i + 1; j < variations.length; j++) {
      const v1 = variations[i];
      const v2 = variations[j];
      
      // Diferença em posição do produto (horizontal)
      const xDiff = Math.abs(v1.layout.productArea.x - v2.layout.productArea.x);
      const xDiffPercent = xDiff / 1080;
      
      // Diferença em posição do produto (vertical)
      const yDiff = Math.abs(v1.layout.productArea.y - v2.layout.productArea.y);
      const yDiffPercent = yDiff / 1350;
      
      // Diferença em tamanho
      const widthDiff = Math.abs(v1.layout.productArea.width - v2.layout.productArea.width);
      const widthDiffPercent = widthDiff / 1080;
      
      const heightDiff = Math.abs(v1.layout.productArea.height - v2.layout.productArea.height);
      const heightDiffPercent = heightDiff / 1350;
      
      // Pelo menos UMA métrica deve ter >30% diferença
      const maxDiff = Math.max(xDiffPercent, yDiffPercent, widthDiffPercent, heightDiffPercent);
      
      if (maxDiff < threshold) {
        return false; // Variações muito similares
      }
    }
  }
  
  return true;
}
```

**Se validação falhar:** Regenerar variações (máximo 2 tentativas) ou usar fallback determinístico.

### Fluxo de execução

```typescript
// lib/ai/visual-composer/service.ts
export async function composeVariations(
  imageProfile: ImageProfile,
  direction: CreativeDirection,
  signature: VisualSignature,
  campaign: CampaignData
): Promise<CompositionVariants> {
  
  // 1. Gate: campanhas informativas não processam
  if (campaign.content_type === "message") {
    return emptyCompositionVariants();
  }

  // 2. Preparar context para LLM
  const context = {
    image: imageProfile,
    direction,
    signature,
    campaign,
    canvas: { width: 1080, height: 1350 },
  };

  // 3. Chamar LLM (gera 4 variações em 1 chamada)
  const response = await callAI(
    [{ role: "system", content: VISUAL_COMPOSER_SYSTEM_PROMPT }],
    { role: "user", content: JSON.stringify(context) }
  );

  // 4. Parse e validação
  const parsed = parseJsonFirstObject(response);
  const validated = CompositionVariantsSchema.safeParse(parsed);

  if (!validated.success) {
    // Fallback: gerar variações determinísticas
    return generateFallbackVariations(context);
  }

  const variations = validated.data.variations;

  // 5. Validar distinctness
  if (!validateDistinctness(variations)) {
    // Re-tentar 1 vez ou usar fallback
    return generateFallbackVariations(context);
  }

  // 6. Validar bounds e overlap
  for (const v of variations) {
    if (!validateBounds(v.layout.productArea)) {
      throw new Error("INVALID_LAYOUT_BOUNDS");
    }
    // ... validar outras áreas
  }

  return {
    direction,
    variations,
    canvas: { width: 1080, height: 1350 },
    generated_at: new Date().toISOString(),
  };
}
```
