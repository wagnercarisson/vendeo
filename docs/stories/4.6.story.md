# Story 4.5: Visual Signature Integration

**Epic:** Epic 4 — Motor de Composição Visual v2.0  
**Story ID:** 4.5  
**Sprint:** TBD  
**Priority:** High  
**Points:** 3  
**Effort:** 2-3 dias  
**Status:** Draft  
**Type:** Feature — Integration

---

## User Story

**Como** lojista,  
**Quero** que minhas artes geradas tenham as cores e identidade visual da minha loja,  
**Para** que haja consistência e reconhecimento da marca.

---

## Public Impact

**Has Impact:** Yes

**User Benefit:** Artes geradas agora incluem cores da loja (primary/secondary) e logo, criando consistência visual entre campanhas. Lojista reconhece "identidade da marca" nas artes.

**Breaking Changes:** None (melhoria sobre sistema atual)

**Migration Path:** Lojas sem Visual Signature definida continuam usando cores padrão.

---

## Objective

Integrar o sistema de **Visual Signatures** (criado na Fase 1) com o Renderer, aplicando `primary_color`, `secondary_color`, `logo_url` e `store_name_typography` nas artes geradas.

---

## Scope IN

- Buscar Visual Signature da loja ao gerar campanha
- Aplicar cores primárias/secundárias no background e elementos decorativos
- Posicionar logo da loja (se disponível) no canto da arte
- Usar tipografia personalizada para nome da loja (se definida)
- Garantir contraste adequado (text vs background) via algoritmo WCAG

## Scope OUT

- Não criar UI de configuração de Visual Signature (já existe em Fase 1)
- Não implementar Context Profiles (Story 4.6)
- Não implementar learning (Story 4.8)

---

## Acceptance Criteria

```gherkin
GIVEN loja com Visual Signature definida (primary_color, logo_url)
WHEN campanha for gerada
THEN arte final usa primary_color no background
AND logo aparece no canto superior direito (80x80px)
AND cores respeitam contraste WCAG AA (4.5:1)
```

```gherkin
GIVEN loja sem Visual Signature definida
WHEN campanha for gerada
THEN sistema usa cores padrão (indigo/purple)
AND logo não é renderizado
```

## Technical Details

```typescript
// Buscar Visual Signature
const signature = await getVisualSignature(storeId);

// Aplicar no Renderer
const artUrl = await renderComposition(spec, productImage, {
  primaryColor: signature.primary_color,
  secondaryColor: signature.secondary_color,
  logoUrl: signature.logo_url,
  typography: signature.store_name_typography,
});
```

### Contrast Validation

```typescript
function ensureContrast(textColor: string, bgColor: string): string {
  const ratio = calculateContrastRatio(textColor, bgColor);
  if (ratio < 4.5) {
    return adjustColorForContrast(textColor, bgColor);
  }
  return textColor;
}
```
