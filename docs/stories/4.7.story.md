# Story 4.6: Context Profiles — Aplicação

**Epic:** Epic 4 — Motor de Composição Visual v2.0  
**Story ID:** 4.6  
**Sprint:** TBD  
**Priority:** High  
**Points:** 5  
**Effort:** 3-4 dias  
**Status:** Draft  
**Type:** Feature — Smart Behavior

---

## User Story

**Como** lojista criando campanha promocional ou sazonal,  
**Quero** que o sistema ajuste automaticamente o visual (cores, intensidade, badges),  
**Para** que a arte reflita o contexto (ex: Natal, Black Friday, produto premium).

---

## Public Impact

**Has Impact:** Yes

**User Benefit:** Sistema ajusta automaticamente visual da arte baseado no contexto (ex: promoção ganha badges de desconto, campanha de Natal usa cores temáticas, produto premium tem design minimalista). Lojista não precisa configurar nada.

**Breaking Changes:** None (melhoria sobre sistema atual)

**Migration Path:** Campanhas sem `visual_context` usam profile "standard" (comportamento atual).

---

## Objective

Integrar os **5 Context Profiles** (standard, promotional, seasonal, premium, urgency) no Intent Resolver e Renderer, ajustando automaticamente mood, intensity e elementos decorativos baseados em `campaigns.visual_context`.

---

## Scope IN

- Implementar aplicação de Context Profiles no Intent Resolver
- Ajustar Renderer para aplicar regras de cada profile:
  - **promotional:** badges de desconto, cores vibrantes, intensity=strong
  - **seasonal:** paleta temática (ex: vermelho/verde para Natal), decorativos sazonais
  - **premium:** minimal design, cores sóbrias, intensity=minimal
  - **urgency:** badges "ÚLTIMAS UNIDADES", cores de alerta, intensity=strong
  - **standard:** balanced, sem ajustes especiais
- Buscar `visual_signature_profiles` da loja e aplicar `composition_rules`
- Criar biblioteca de themes sazonais pré-calibrados (Natal, Páscoa, Black Friday)

## Scope OUT

- Não criar UI para configurar profiles (já existe schema em Fase 1)
- Não implementar aprendizado de preferências (Story 4.8)

---

## Acceptance Criteria

```gherkin
GIVEN campanha com visual_context="promotional"
WHEN Intent Resolver gerar CreativeDirection
THEN priceEmphasis === "high"
AND visualIntensity === "strong"
AND Renderer adiciona badge "50% OFF"
```

```gherkin
GIVEN campanha com visual_context="seasonal" e theme="Natal"
WHEN artes forem geradas
THEN paleta de cores inclui vermelho/verde
AND elementos decorativos incluem tema natalino (opcional)
```

```gherkin
GIVEN campanha com visual_context="premium"
WHEN artes forem geradas
THEN visualIntensity === "minimal"
AND sem badges chamativos
AND tipografia sofisticada
```

## Technical Details

```typescript
// Buscar profile da loja
const profile = await getVisualProfile(storeId, visual_context);

// Aplicar no Intent Resolver
const direction = await resolveIntent(imageProfile, campaignData, {
  visualSignature: signature,
  contextProfile: profile,
});

// Renderer ajusta baseado no profile
if (profile.context_type === "promotional") {
  addPromotionalBadge(ctx, "50% OFF");
  increaseColorSaturation(colors, 1.3);
}
```
