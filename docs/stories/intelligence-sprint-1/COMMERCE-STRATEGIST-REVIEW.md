### 2. Loja de bebidas ⚠️ APPROVED WITH NOTES

**Visual Style:** "sophisticated and inviting"  
**Colors:** `#8B0000` (burgundy), `#D4AF37` (gold), `#2E7D32` (dark green)

**✅ Strengths:**
- Burgundy/gold evoca sofisticação (bom para adegas premium)
- "Wine glass" e "grape cluster" são icônicos
- Avoid "cartoonish drinks" é apropriado

**⚠️ Concerns:**
- Template parece muito focado em **vinho/bebidas alcoólicas**
- Lojas de bebidas brasileiras vendem refrigerantes, energéticos, sucos (não só álcool)
- "Grape cluster" é específico demais para vinho

**💡 Recommendation:**
- Adicionar "bottle silhouette" como elemento icônico mais genérico
- Considerar incluir "refreshment" no prompt base para ampliar escopo
- **Ação:** Ajustar prompt para ser mais inclusivo (refrigerantes, energéticos)

## 🔧 Recommended Adjustments

### Priority 1: Immediate (Before Story 3 Implementation)

#### 1.1 Loja de bebidas - Ampliar escopo

**Current Issue:** Muito focado em vinho/bebidas alcoólicas

**Suggested Fix:**
```typescript
basePrompt:
  'A minimalist, professional logo for "{storeName}", a beverage store. 
  Style: sophisticated and inviting. Design: Simple geometric shapes 
  representing a bottle, glass, or refreshment symbol, maximum 2-3 colors 
  (burgundy, amber, or dark green), no text, flat design suitable for 
  social media, white background, high contrast. The logo should evoke 
  quality, refreshment, and social enjoyment.',

iconicElements: [
  "bottle silhouette",        // Mais genérico
  "glass outline",            // Removido "wine"
  "droplet or wave",          // Adicionar elemento de refreshment
  "barrel icon",              // Mantido
],
```

**Impact:** Aumenta relevância para lojas de bebidas mistas (refrigerantes + álcool)

---

### Priority 2: Recommended (Post-MVP)

#### 2.1 Materiais de construção - Aquecer paleta

**Current Issue:** Cinza dominante pode parecer frio

**Suggested Fix:**
```typescript
colorSuggestions: ["#FF6F00", "#607D8B", "#D84315"],
// Reordenado: Orange → Gray → Terra (era Gray → Orange → Blue)
```

**Impact:** Logo mais quente e alinhado com setor de construção brasileiro

---

#### 2.2 Casa & Decoração - Suavizar teal

**Current Issue:** Teal muito saturado para "cozy"

**Suggested Fix:**
```typescript
colorSuggestions: ["#795548", "#80CBC4", "#FF7043"],
// Teal suavizado: #009688 → #80CBC4 (mais muted)
```

**Impact:** Logo mais acolhedor e cozy

---

### Priority 3: Future Enhancement

#### 3.1 "Outro…" - Sub-segmentos

**Suggestion:** Criar prompts adicionais quando usuário escolhe "Outro…"

**Workflow:**
1. Usuário escolhe "Outro…"
2. Modal pergunta: "Qual tipo de negócio? (Ex: Ótica, Livraria, Floricultura)"
3. Input livre → GPT-4 gera prompt customizado baseado em descrição
4. Fallback para template genérico se GPT-4 falhar

**Impact:** Melhora qualidade de logos para segmentos não-mapeados
