## 🎯 VISÃO ESTRATÉGICA

O Vendeo necessita de um **squad de marketing especializado** que trabalhe junto ao motor visual para gerar campanhas que **realmente convertem** em adegas e mercearias.

**Meta:** Posicionar Vendeo como o **"braço direito" do lojista de bairro** — ferramenta que o ajuda a vender mais através de social.

---

### Por Quê?

## 🏗️ COMO O SQUAD FUNCIONA

### Fluxo de Campanha (Simplificado)

```
LOJISTA ENTRA DADOS
    ↓
@commerce-strategist: "É sexta? É happy hour?"
    ↓
@prompt-eng: "Qual prompt usar pra este contexto?"
    ↓
[IA GERA COPY]
    ↓
@content-copy: "Esse CTA converte? Tone certo?"
    ↓
@brand-designer: "Qual layout destaca a marca? Cliente reconhece imediatamente?"
    ↓
[MOTOR VISUAL GERA 3 ARTES]
    ↓
@ux-design: "UI clara? Visual OK?"
    ↓
LOJISTA VÊ 3 OPÇÕES E ESCOLHE
```

**Tempo Total:** 2 minutos (ideal: do celular, enquanto abastece loja)

## 📈 IMPACTO ESPERADO (POR LOJISTA)

### Antes do Vendeo
- 100 clientes base (bairro)
- Social media: Caótica ou inexistente
- Churn: -30-40%/mês (perda pra concorrência)
- Crescimento: ZERO
- Faturamento: Base R$ 50k/mês

### Depois do Vendeo (3 meses)
- 100+ clientes base (retenção melhorada)
- Social media: 3-5 campanhas/semana (organizado)
- Churn: -10-15%/mês (brand recall +30%)
- Crescimento: +15-25% clientes novos/mês
- Ticket médio: +10-15% (impulso)
- **Faturamento: R$ 62.5-67.5k/mês (+30% líquido) ✅**

## 🎯 RESPOSTA FINAL ÀS SUAS PERGUNTAS INICIAIS

### 1. Quais agentes de marketing são necessários?

**5 Core Agents:**
- @prompt-eng (otimizar IA para adegas)
- @commerce-strategist (NEW — contexto comercial)
- @content-copy (NEW — copywriting que converte)
- @brand-designer (NEW — identidade visual única)
- @ux-design (validar UX)

### 2. O que cada um deve fazer?

Veja documento "Squad Structure" — cada um tem 3-4 responsabilidades específicas.

### 3. Quais integrar ao projeto Vendeo?

Todos os 5. Funcionam em paralelo durante geração de campanha.

### 4. Como compor campanhas personalizadas + padronizadas?

- **Personalizadas:** Cada campanha herda Visual Signature única da loja
- **Padronizadas:** Mesma qualidade, mesma estratégia, mesma excelência em TODAS campanhas

### 5. Como garantir retorno (vendas)?

- Selecionar adegas como segmento (alto giro, ROI claro)
- Calibrar squad com contexto comercial real (dia, hora, clima, sazonalidade)
- Medir feedback (survey) → Alimentar IA com dados de sucesso
- Premium tier com IA orquestradora = automação que aprende
