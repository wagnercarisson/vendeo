### "Por que Vendeo não é inviabilizado por IA gratuita?"
```
1. 80% lojistas ainda NÃO usam IA bem
2. ChatGPT leva 30 min, Vendeo leva 2 min (15x diferença)
3. ChatGPT gera genérico, Vendeo gera contextual
4. Vendeo resolve TEMPO + ESTRATÉGIA, não ferramenta
```

### "Qual é a estratégia de plans?"
```
FREE: Logo IA básica + Tier 1 + 5 campanhas
      (Meta: Usuário vê valor)

BASIC: Logo IA profissional + Tier 1+2 + 50 campanhas
       (Meta: Usuário sério)

PRO: Logo premium + Tier 1+2+3 + analytics + unlimited
     (Meta: "Vendeo é core do meu marketing")

Target upgrades: 15-20% Free→Basic, 10-15% Basic→Pro
```

### "Quais são os diferenciadores defensáveis?"
```
1. Contexto automático (seu horário, público, desafio)
2. Integração 1-click (publica direto IG/WA)
3. Analytics + feedback ("Funcionou!")
4. Aprendizado passivo (IA melhora a cada campanha)
5. Garantia de resultado ("Se não vender, reembolso")
6. Comunidade (vê o que funciona em outras lojas)
```
