## 📋 REQUIREMENTS

### **1. Prompt Section Structure**

Create `<badge_generation>` section with:

```xml
<badge_generation>
  <purpose>
    Generate short, compelling badge and CTA texts based on campaign intent
  </purpose>

  <rules>
    - [Your rules here]
  </rules>

  <contexts>
    - promotional: [definition + examples]
    - urgency: [definition + examples]
    - premium: [definition + examples]
    - standard: [definition + examples]
  </contexts>

  <output_format>
    {
      "badgeText": "string | null",
      "badgePosition": "top-left" | "top-center" | "top-right",
      "ctaText": "string | null"
    }
  </output_format>

  <fallback_trigger>
    When to use fallback templates instead of AI generation
  </fallback_trigger>
</badge_generation>
```

### **2. Rules to Include**

**Badge Text Rules:**
- Max 30 characters (strict)
- Product-specific when possible
- Avoid generic jargon ("MEGA OFERTA", "COMPRE AGORA")
- Use discount/price info when available
- Tone matches context (urgency = action verbs, premium = exclusivity, etc.)

**CTA Text Rules:**
- Max 25 characters (strict)
- Action-oriented verbs
- Contextual (promotional ≠ premium)
- Product-aligned when possible

**Position Rules:**
- promotional → top-left (high visibility)
- urgency → top-center (attention)
- premium → top-right (subtle elegance)
- standard → top-center (neutral)

### **3. Fallback Integration**

Define when AI should fallback to templates:

**Trigger conditions:**
- AI output exceeds character limits
- AI output is generic/jargon-filled
- AI fails to generate (retry once, then fallback)
- Campaign instructions empty/unclear

**Fallback source:** `docs/FALLBACK-TEMPLATES-MOTOR2-PHASE1.md`

### **4. Examples (Minimum 10)**

Provide input/output examples covering:
- 5 contexts (promotional, urgency, premium, standard, seasonal)
- 3 badge types (promotional badges, urgency badges, CTAs only)
- Edge cases (empty instructions, too long output, fallback trigger)

**Example format:**
```
INPUT:
- Product: Chocolate Nestlé 150g
- Price: R$ 8.50 → R$ 6.40
- Campaign Instructions: "Promoção de Páscoa com 25% de desconto"
- Context: promotional

OUTPUT:
{
  "badgeText": "25% OFF Páscoa",
  "badgePosition": "top-left",
  "ctaText": "Aproveite agora"
}
```
